# 中钾华强官网实施记录

## 2026-06-16 兴发集团官网逻辑参考调整

参考对象：https://www.xingfagroup.com/index.php/Welcome/

本次调整不复制具体视觉素材与内容，只学习其企业官网信息架构逻辑：

- 顶部工具栏 + 主导航：顶部放确认说明、联系电话、语言入口；主导航保持公司栏目。
- 首屏大横幅：用大图建立化工企业与綦江产业基地形象，避免左右分栏卡片化首屏。
- 企业概况：以公司定位、项目所在地、展示产品数和綦江区位图承接可信度。
- 产品中心：按横向产品分类入口展示，只保留当前确认的七个产品。
- 技术与质量：保留为企业站必需的质量、工艺和安全环保说明入口。
- 新闻与资料：作为公司动态、检测报告、产品手册和下载文件入口。

未采用内容：

- 投资者关系、招标中心、招聘加入等模块当前无真实内容支撑，首页不做。
- 不新增未确认产品、不展示未确认产能、不写未经确认的客户、资质或收益承诺。

视觉原则：

- 色彩继续沿用简洁绿色、白色和浅绿色，不使用金色、暖白或复杂渐变。
- 动效只保留首屏大图轻微呼吸、卡片 hover、滚动 reveal，避免过度炫技。
- 移动端采用横向栏目导航，保证栏目直接可见。
